# FSBO Hunter — Southern California

## Status
Basic scraping is blocked by Zillow, ForSaleByOwner.com (403 errors). Need different approach.

## What Works

### Free Methods (manual + alerts)
1. **Zillow FSBO alerts** — Set up saved search with FSBO filter at zillow.com, get email alerts
2. **Craigslist RSS** — Monitor OC Craigslist real estate section for FSBO posts
3. **Facebook Marketplace** — Search "house for sale by owner" in Orange County
4. **Nextdoor** — FSBO posts from neighbors

### Paid Data Options (for automation)
1. **PropStream** ($99/mo) — FSBO lists, absentee owners, pre-foreclosures, motivated sellers
2. **BatchLeads** ($79/mo) — Skip tracing, FSBO data, property data
3. **CompCurve** — 40-60K daily FSBO records via API
4. **ATTOM Data** — Property data API with ownership info

### What the Bot CAN Do
- Monitor Craigslist RSS feeds for FSBO keywords
- Alert on price drops for watched properties
- Score leads based on criteria (price, location, days on market)
- Auto-organize leads into a dashboard
- Send daily summary of new FSBO listings
